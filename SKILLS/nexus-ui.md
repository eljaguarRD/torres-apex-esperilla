# SKILL: Nexus-UI (Glassmorphism)

## Description
This skill provides standardized CSS and components for the Nexus CRM "AntiGravity" aesthetic.

## Patterns
- **Glassmorphism**: `backdrop-filter: blur(10px); background: rgba(255, 255, 255, 0.1); border: 1px solid rgba(255, 255, 255, 0.2);`
- **Gradients**: Vibrant purples and blues (`#6a11cb` to `#2575fc`).
- **Typography**: Inter or Montserrat.

## Usage
Call this skill when creating new frontend modules to ensure consistency.
